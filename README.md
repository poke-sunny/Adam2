# Adam Ellahi Project F1

Premium motorsport digital ecosystem for Adam Ellahi.

## Contents

- `website` - Hostable Next.js public website.
- `design-system` - Formula One Digital Design Language working asset library.
- `Project_F1_Design_Bible_v1` - Creative, UX, UI, sponsor, motion, AI, and developer specifications.
- `source` - Imported source creative pack.

## Local Website

```bash
cd website
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

Deploy the `website` folder as a Next.js app on Vercel.
