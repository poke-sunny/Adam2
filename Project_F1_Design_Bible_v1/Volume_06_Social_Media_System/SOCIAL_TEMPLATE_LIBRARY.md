# Volume 6 - Social Media System

One story should become unlimited outputs.

A single race weekend should create:

- Instagram posts.
- Instagram stories.
- Reels.
- TikTok videos.
- YouTube Shorts.
- YouTube long-form.
- LinkedIn posts.
- X posts.
- Threads posts.
- Facebook posts.
- Blog.
- Newsletter.
- Sponsor report.
- Media kit update.
- Press release.

## Format Requirements

- Instagram square: 1080 x 1080.
- Instagram portrait: 1080 x 1350.
- Story/Reel/TikTok/Shorts: 1080 x 1920.
- YouTube thumbnail: 1280 x 720.
- LinkedIn image: 1200 x 627.
- X image: 1600 x 900.
- Facebook post: 1200 x 630.

## Visual System

Use:

- Carbon black backgrounds.
- Strong photography.
- Titanium labels.
- Electric Cyan telemetry accents.
- British Racing Green for milestones and sponsor credibility.
- Mono numbers for results.

Avoid:

- Generic racing flames.
- Fake smoke overlays.
- Loud gradient poster design.
- Text-heavy story slides.

## Copy Rules

Each post needs one clear job:

- Inform.
- Invite.
- Prove.
- Reflect.
- Thank.
- Convert.

Use short captions with specific proof. Let the image carry emotion.
