# Volume 10 - AI Prompt Library

## How To Use This Library

Use prompts as production briefs, not vague inspiration. Each generated asset should be reviewed against the Design Bible before being accepted.

Rules:

- Never generate official Formula One logos, team liveries, or protected brand assets.
- Keep Adam Ellahi branding original.
- Use no text in generated images unless the prompt requires exact copy.
- Prefer generous negative space for UI images.
- Generate desktop and mobile crops separately.
