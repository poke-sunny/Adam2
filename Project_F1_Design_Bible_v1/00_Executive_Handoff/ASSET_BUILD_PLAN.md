# Asset Build Plan

This plan turns the Design Bible into a complete 5-10 GB design asset kit over production cycles.

## Current v1.0 Deliverable

This version creates:

- Ten-volume Design Bible.
- Working design-system folder.
- Logo starter system in SVG.
- Favicon and app icon starter SVGs.
- Motorsport icon starter sprite.
- Background texture starter SVGs.
- Design tokens in JSON and CSS.
- AI builder prompt.
- Image generation prompt library.
- UI screen matrix and production counts.
- Sponsor, Mission Control, social, photography, and motion specs.

## Target Full Kit

The full agency-grade kit should contain:

- 250-500 original UI screens.
- 300+ component states.
- 200-400 bespoke motorsport icons.
- 100+ social templates.
- 40+ photography mood boards.
- 30+ logo and app icon exports.
- 20+ background and texture families.
- 30+ motion and Lottie concepts.
- Developer tokens for CSS, Tailwind, Figma, and app code.
- AI prompt library for continuous generation.

## Production Phases

### Phase 1 - Identity Lock

Deliver:

- Logo suite.
- Helmet mark.
- App icon.
- Favicon set.
- Colour tokens.
- Typography rules.
- Brand voice.
- Photography direction.

Exit criteria:

- Logo works at 16 px, 44 px, 120 px, race suit scale, and full desktop masthead scale.
- Colour contrast passes WCAG AA.
- AI builder can create a first homepage without guessing the brand.

### Phase 2 - UI System

Deliver:

- Buttons.
- Cards.
- Navigation.
- Forms.
- Tables.
- Tabs.
- Modals.
- Data cards.
- Sponsor cards.
- Race cards.
- Gallery and video components.
- AI assistant components.

Exit criteria:

- All components include mobile, desktop, hover, focus, disabled, loading, empty, and error states.

### Phase 3 - Public Platform Screens

Deliver:

- Core public site screens.
- Newsroom.
- Media centre.
- Results and timeline.
- Sponsor conversion pages.
- STEM and community pages.

Exit criteria:

- Every page answers: why Adam, why now, why invest, why trust, why follow.

### Phase 4 - Enterprise Products

Deliver:

- Mission Control.
- Sponsor Portal.
- Team Portal.
- AI Console.
- Commercial pipeline.
- Revenue dashboard.

Exit criteria:

- A sponsor can understand exposure, ROI, next actions, and campaign value without a meeting.

### Phase 5 - Content Engine

Deliver:

- Social templates.
- Press kit.
- Race weekend content pipeline.
- AI content prompts.
- Media reports.
- Sponsor reports.

Exit criteria:

- One race weekend story can become posts, reports, press releases, blogs, newsletters, and sponsor updates.

## Naming Rules

Use lowercase folders for exported production assets where possible:

- `logo-ae-mark-dark.svg`
- `logo-horizontal-light.svg`
- `icon-helmet.svg`
- `texture-carbon-fibre-01.webp`
- `screen-home-mobile-01.png`
- `social-instagram-post-race-result-01.fig`

Keep source files editable and exports deterministic.
