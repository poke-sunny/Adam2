# Volume 8 - Motion And Animation

## Motion Principle

Motion should communicate performance and intelligence. It should never feel like decoration added after the fact.

## Motion Personality

- Fast but controlled.
- Precise.
- Calm.
- Mechanical.
- Responsive.
- Premium.

## Timing System

- Micro interaction: 120 ms.
- Button press: 90 ms.
- Tooltip: 120 ms in, 80 ms out.
- Card hover: 180 ms.
- Drawer open: 240 ms.
- Modal open: 260 ms.
- Page transition: 320 ms.
- Chart reveal: 500-900 ms depending on density.
- Counter animation: 700-1200 ms.

## Easing

Use custom cubic bezier curves:

- Standard: `cubic-bezier(0.2, 0, 0, 1)`.
- Fast out: `cubic-bezier(0.16, 1, 0.3, 1)`.
- Mechanical: `cubic-bezier(0.65, 0, 0.35, 1)`.

## Reduced Motion

If reduced motion is enabled:

- Disable large page transitions.
- Replace parallax with static positioning.
- Use opacity changes instead of movement where possible.
- Keep essential state changes visible.

## Do Not Use

- Bouncy cartoon easing.
- Excessive parallax.
- Constant particle decoration.
- Loud glow pulses.
- Animations that delay reading or action.
