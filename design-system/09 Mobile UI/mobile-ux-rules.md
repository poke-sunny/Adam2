# Mobile UX Rules

Mobile is the primary sponsor discovery experience.

## Mobile Priorities

- Adam identity.
- Current proof.
- Next race or latest result.
- Sponsor CTA.
- Story preview.
- Media assets.
- Contact path.

## Rules

- Design from 390 px width first.
- Keep primary CTAs thumb reachable.
- Use 44 px minimum touch targets.
- Avoid multi-column layouts.
- Collapse tables into cards.
- Keep dashboards summary-first.
- Never hide sponsor inquiry behind a deep menu.
- Keep text inside containers at all widths.
- Avoid sticky elements that cover content.

## Mobile Product Modes

Public site:

- Story-led.
- Fast.
- Cinematic.
- Sponsor CTA visible.

Mission Control:

- Alert-led.
- Today view.
- Saved views.
- Drill-down metrics.

Sponsor Portal:

- Executive summary first.
- Download report.
- Book review call.
- View activation opportunities.
