The full system should include 100+ editable social layouts.

- Race weekend announcement.
- Practice recap.
- Qualifying result.
- Race result.
- Track guide.
- Result comparison.
- Behind the visor.
- Training day.

- Video thumbnail.
- Story countdown.
- Lessons learned.

## Export Sizes

- Instagram square: 1080 x 1080.
- Instagram portrait: 1080 x 1350.
- Story/Reel/TikTok/Shorts: 1080 x 1920.
- YouTube thumbnail: 1280 x 720.
- LinkedIn image: 1200 x 627.
- X image: 1600 x 900.
- Facebook post: 1200 x 630.

## Design Rules

- Strong image first.
- Minimal copy.
- Mono numbers.
- Carbon black base.
- Titanium labels.
- Green for performance or partner authority.
- Cyan for live telemetry or AI.
