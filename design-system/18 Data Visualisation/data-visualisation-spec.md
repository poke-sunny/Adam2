# Data Visualisation Specification

## Required Chart Types

- Line.
- Area.
- Bar.
- Stacked bar.
- Heatmap.
- Radar.
- Geo map.
- Scatter.
- Timeline.
- Funnel.
- Gauge only when absolutely necessary.
- Telemetry trace.
- Race pace chart.
- Audience growth chart.
- Sponsor ROI chart.
- Revenue chart.

## Chart Style

- Dark panels.
- Soft graphite grid lines.
- White labels.
- Titanium secondary labels.
- British Racing Green for positive progress.
- Electric Cyan for live telemetry and AI.
- Warning and danger colours only for real risk.

## Chart Requirements

Every chart must include:

- Title.
- Period.
- Source.
- Tooltip.
- Empty state.
- Plain-English insight.

## Do Not Use

- Unlabeled axes.
- 3D chart effects.
- Decorative gradients that hide values.
- Colour-only meaning.
- Vanity metrics without business context.
