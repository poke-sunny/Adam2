# Illustration Pack Index

## Purpose

Illustrations explain systems, journeys, and technical concepts that photography cannot show clearly.

## Required Illustration Families

- Track illustrations.
- Garage.
- Pit wall.
- Telemetry.
- AI assistant.
- Mission Control.
- Mechanics.
- Sponsors.
- Fans.
- Global map.
- Analytics.
- Race weekend flow.
- Sponsor value chain.
- Content pipeline.

## Style Rules

- Technical, restrained, and precise.
- Use line, grid, and data-inspired forms.
- Avoid cartoon treatment.
- Avoid copying official motorsport graphics.
- Use carbon, titanium, green, and cyan palette.

## Export Rules

- SVG for simple illustrations.
- WebP/AVIF for complex rendered scenes.
- Include light and dark variants only when necessary.
