# Dashboard Pack Index

## Purpose

Define dashboard patterns used across Mission Control, Sponsor Portal, AI Console, Media Centre, CRM, analytics, and fan intelligence.

## Required Dashboard Families

- Mission Control.
- Sponsor Portal.
- AI Console.
- Media Centre.
- Analytics.
- CRM.
- Fan Intelligence.
- Revenue.
- Content Performance.
- Commercial Pipeline.

## Required Breakpoints

Each dashboard family needs:

- Desktop.
- Tablet.
- Mobile summary.
- Empty state.
- Loading state.
- Alert state.

## Shared Modules

- KPI tile.
- Trend card.
- Chart panel.
- Insight card.
- Recommendation card.
- Alert row.
- Activity feed.
- Data table.
- Report builder.
- Download centre.
- AI assistant panel.

## Rules

- Dashboards must be dense but calm.
- Every number needs context.
- Every dashboard should produce a next action.
- Do not display vanity metrics without explanation.
- Use Electric Cyan for live data and AI.
- Use British Racing Green for positive progress and commercial value.
