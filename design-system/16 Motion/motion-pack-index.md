# Motion Pack Index

## Purpose

Motion communicates performance, responsiveness, and intelligence.

## Required Motion Patterns

- Page transitions.
- Card hover.
- Graph animation.
- Counter animation.
- Logo reveal.
- Speed blur.
- Telemetry pulse.
- Particle drift only when purposeful.
- Scroll reveal.
- Button animation.
- Glass glow.
- Drawer open.
- Modal open.
- Report generation.
- AI thinking.

## Timing

- Press: 90 ms.
- Micro: 120 ms.
- Hover: 180 ms.
- Drawer: 240 ms.
- Modal: 260 ms.
- Page: 320 ms.
- Chart: 500-900 ms.

## Rules

- Motion must clarify state.
- Reduced motion must be supported.
- Avoid bounce.
- Avoid constant ambient movement.
