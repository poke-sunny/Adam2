# Photography Pack Index

## Purpose

Photography is the emotional proof layer of the platform.

## Required Folders

Create production folders for:

- Hero.
- Portrait.
- Garage.
- Pitlane.
- Helmet.
- Emotion.
- Rain.
- Victory.
- Failure.
- Training.
- Gym.
- Simulator.
- Travel.
- Sponsors.
- Fans.
- Lifestyle.
- Family.
- Engineering.
- Mechanics.
- Night.
- Drone.
- Macro.
- Cockpit.
- Hands.
- Eyes.
- Tyres.
- Brakes.
- Carbon.
- Details.

## Required Exports

Each accepted image should have:

- Original.
- Desktop hero crop.
- Mobile hero crop.
- Social crop.
- WebP or AVIF production export.
- Alt text.
- Usage notes.

## Rules

- Do not use generic stock racing imagery.
- Do not include unlicensed logos.
- Use negative space for UI overlays where needed.
- Show journey, discipline, recovery, and learning.
