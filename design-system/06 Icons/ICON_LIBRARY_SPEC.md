`design-system/06 Icons/svg/motorsport-icon-sprite.svg`

```html

  <use href="/icons/motorsport-icon-sprite.svg#ae-icon-helmet"></use>
</svg>
```

## Style Rules

- 24 x 24 viewBox.
- 1.8 px stroke.
- Rounded caps and joins where appropriate.
- No filled decorative bulk except small emphasis dots.
- Use `currentColor`.
- Every icon must work at 16 px and 24 px.
- Avoid generic clip-art motorsport shapes.
