# Favicon Export Checklist

- `svg/favicon.svg`
- `svg/app-icon.svg`

## Required Exports

- 16 x 16 PNG.
- 24 x 24 PNG.
- 32 x 32 PNG.
- 48 x 48 PNG.
- 64 x 64 PNG.
- 128 x 128 PNG.
- 180 x 180 Apple touch icon.
- 192 x 192 Android icon.
- 256 x 256 PNG.
- 512 x 512 PWA icon.
- Safari pinned tab SVG.
- Windows tile icon.
- `favicon.ico`.

## Rules

- Use the compact AE mark.
- Keep contrast high.
- Avoid thin details that disappear at 16 px.
- Test on dark browser tabs and light browser tabs.
