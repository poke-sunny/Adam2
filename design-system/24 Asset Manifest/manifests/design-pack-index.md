# 22 Design Pack Index

This index maps the complete Formula One Digital Design Language pack structure.

## Packs

1. `01 Brand` - F1DL summary, brand rules, voice, positioning.
2. `02 Logos` - desktop, mobile, helmet, suit, app, favicon marks.
3. `03 Favicons` - favicon and platform icon exports.
4. `04 Colours` - colour system and usage rules.
5. `05 Typography` - font system and hierarchy.
6. `06 Icons` - bespoke motorsport icon library.
7. `07 Components` - component checklist and state rules.
8. `08 Desktop UI` - desktop screen matrix.
9. `09 Mobile UI` - mobile UX rules.
10. `10 Dashboard` - shared dashboard system.
11. `11 Mission Control` - operating dashboard pack.
12. `12 Sponsor Portal` - sponsor experience pack.
13. `13 Photography` - photography folders and shot direction.
14. `14 Illustrations` - technical illustration system.
15. `15 Backgrounds` - background and texture system.
16. `16 Motion` - motion patterns and timings.
17. `17 Animations` - Lottie concepts.
18. `18 Data Visualisation` - chart and data rules.
19. `19 Social Media` - social template index.
20. `20 Marketing` - marketing page and campaign pack.
21. `21 AI` - AI product and component system.
22. `22 Design Tokens` - JSON, CSS, and developer tokens.

Additional handoff packs:

- `23 AI Handoff` - prompts for AI builders and asset generation.
- `24 Asset Manifest` - package status, inventories, and export rules.

## Rule

Every folder must be treated as a live product area. If a future build adds assets, update the relevant pack index and the master asset manifest.
