# Lottie Pack Index

## Purpose

Lottie animations should provide lightweight, premium motion for repeatable UI moments.

## Required Lottie Concepts

- AE logo line reveal.
- Telemetry pulse.
- Track map trace.
- Race countdown ring.
- Sponsor report assembly.
- AI insight scan.
- Data sync.
- Media coverage radar.
- Global audience map pulse.
- Upload complete.
- Download complete.
- Alert acknowledged.

## Export Requirements

- JSON.
- Transparent background.
- Dark mode safe.
- Reduced motion fallback.
- Looping only when useful.
- Under 150 KB where practical.

## Rules

- No playful bounce.
- No distracting infinite loops.
- No complex animation where CSS is enough.
