# Typography System

## Fonts

Display:

- Space Grotesk.

Body:

- Inter.

Numbers and telemetry:

- IBM Plex Mono.

## Weights

- 300 light.
- 400 regular.
- 500 medium.
- 600 semibold.
- 700 bold.
- 800 extrabold.

## Rules

- Use Space Grotesk for headlines and high-value labels.
- Use Inter for body, navigation, forms, and dense dashboard UI.
- Use IBM Plex Mono for statistics, timings, telemetry, and race results.
- Letter spacing should remain 0 except for rare technical labels.
- Do not scale type using viewport width.
- Use smaller, tighter headings inside dashboard panels.

## Hierarchy

Hero:

- 48-60 px desktop.
- 36-44 px tablet.
- 32-40 px mobile.

Page heading:

- 36-48 px desktop.
- 28-36 px mobile.

Section heading:

- 24-32 px desktop.
- 22-28 px mobile.

Dashboard heading:

- 16-22 px.

Body:

- 16-18 px editorial.
- 14-16 px dashboard.

Meta:

- 12-14 px.
